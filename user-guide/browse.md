# Browse

## Searching

1. turn on layer
2. click options button
3. select Table
4. click Search
5. start typing text
6. select desired record

![](./img/table-view-search.png){style="width:400px"}

The map will zoom to the selected record's location, highlight the feature, and hide all other features.
