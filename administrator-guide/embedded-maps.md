title: Embedded Maps
---

Customised Pozi maps can be embedded in your organisationâ€™s web pages using a standard HTML iframe. Copy the link of the desired map from your browser's address bar or by clicking the Share button from the menu at the top right. If you want the page to show the default map extent, you can strip out the location parameters (eg `x[144.44500]/y[-36.73075]/z[10]/`). Then wrap the link in an iframe tag.

```
<iframe width="100%" height="500px" frameborder="0" scrolling="no" src="https://bendigo.pozi.com/"></iframe>
```
